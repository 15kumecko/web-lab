# Лабораторна робота №1 — Design Patterns

Мета: ознайомитись з основними шаблонами проєктування та продемонструвати їх практичне застосування.

## 1. Creational: Singleton
Вирішує проблему гарантування наявності лише одного екземпляра класу та глобальної точки доступу до нього.

## 2. Structural: Adapter
Вирішує проблему несумісності інтерфейсів різних класів.

## 3. Behavioral: Observer
Вирішує проблему автоматичного сповіщення залежних об'єктів про зміну стану головного об'єкта.
