# Лабораторна робота №5 — Deployment та CI/CD

## Розгортання
Застосунок контейнеризовано за допомогою Docker та налаштовано для розгортання на платформі Render.

## CI/CD Pipeline (GitHub Actions)
Пайплайн складається з двох етапів:
1. Build and Test: автоматичне підняття оточення, встановлення залежностей та запуск тестів (pytest).
2. Deploy: автоматичний деплой на сервер за допомогою Webhook після успішного проходження тестів.
