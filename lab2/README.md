# Лабораторна робота №2 — REST API

Реалізовано серверну частину застосунку за допомогою Python та FastAPI.

## Сутності та зв'язки
1. **Author** (Автор)
2. **Article** (Стаття) — належить автору (зв'язок 1 до багатьох).
3. **Comment** (Коментар) — належить статті (зв'язок 1 до багатьох). 

При запиті списку статей (`GET /articles`), коментарі повертаються як вкладені об'єкти.

## Функціонал
- **CRUD операції**: реалізовано GET, POST, PUT, DELETE для основних сутностей.
- **HTTP коди**: використовуються 200 (OK), 201 (Created), 204 (No Content), 404 (Not Found).
- **Пагінація**: параметри `limit` та `offset`.
- **Сортування**: параметр `sort_by`.
- **Фільтрація**: параметр `author_id`.

## Запуск та Документація
1. Встановити залежності: `pip install -r requirements.txt`
2. Запустити сервер: `uvicorn main:app --reload`
3. Інтерактивна Swagger документація API автоматично доступна за адресою: `http://127.0.0.1:8000/docs`
