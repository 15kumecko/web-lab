# Лабораторна робота №3 — Frontend-застосунок

Простий клієнтський застосунок (SPA) на Vanilla JS.

## Реалізовані вимоги:
- **3 сторінки**: Авторизація (Login), Головна (Home - список статей), Профіль (Profile).
- **Авторизація**: імітація логіну зі збереженням токена в `localStorage`.
- **Спільний стан (Global State)**: реалізовано через глобальний об'єкт `state` у JS, який містить інформацію про юзера та завантажені статті.
- **Підключення до API**: застосунок робить запити до `http://127.0.0.1:8000/articles` (Лабораторна №2).
- **Обробка помилок**: блок `try/catch` для API запитів із виведенням повідомлення для користувача, якщо сервер недоступний.
- **Навігація**: базова хеш-навігація (`window.location.hash`).

## Як запустити:
Оскільки це звичайні HTML/JS/CSS файли, достатньо відкрити `index.html` у будь-якому браузері (або використати Live Server у VS Code).
